Plan every aspect of a novel or other writing project

== Description ==

Writing a novel is just the beginning. This plugin is a comprehensive tool to help writers plan and organize the many details that must be handled in order to successfully publish a novel. It will be especially useful to authors who plan to self-publish. 

It's a proof-of-concept that WordPress can do much more for writers that just power a basic website. 

The plugin is completely free with no upgrade nags anywhere. Enjoy.


= Features =

* Covers everything required to publish a novel
* Includes an Outline tool to plan all your scenes/chapters
* Includes a Character tool where you can add photos and plan your characters
* Includes a budget that can be exported to CSV
* Includes 5 built-in reports
* Allows you to attach documents, like manuscript drafts, research, etc.
* Unlike desktop software, it can be accessed anywhere
* Uses the power of native WordPress features, including custom post types

For more info, vist the [Plan My Novel home page](http://cato.io/plan-my-novel).

== Installation ==

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
